# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

In short: treat everyone with respect. Assume good intent. Disagree on technical substance without making it personal. If you wouldn't say it on a video call with your team, don't say it here.

## Scope

The covenant applies to all project spaces: issues, pull requests, discussions, commit messages, and any direct communication tied to this repo.

## Reporting

To report unacceptable behavior, email **edchapman88@gmail.com**. Reports are treated as confidential. The repo owner is the sole enforcer and will respond within a reasonable timeframe.

The full text of the covenant — including the enforcement guidelines and the four-tier response framework (correction, warning, temporary ban, permanent ban) — is at the link above. Read it.
